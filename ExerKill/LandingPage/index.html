L'exercice physique : un enjeu crucial pour la santé

La sédentarité est devenue un problème majeur de santé publique. Passer trop de temps assis devant un écran, manger des aliments trop gras et ne pas faire d'exercice régulièrement augmente les risques de maladies. ExerKill vous encourage à bouger tout en vous amusant !

Comment jouer ?

ExerKill utilise votre webcam et la détection de votre main pour contrôler le jeu :

  • Déplacez votre main droite pour contrôler le vaisseau
  • Main près du bord gauche : tournez à gauche
  • Main près du bord droit : tournez à droite
  • Main vers le haut : avancez
  • Appuyez sur ESPACE pour tirer sur les mauvaises habitudes (hamburgers, téléphones)
  • Évitez de tirer sur les bonnes habitudes (exercice, fruits et légumes)
